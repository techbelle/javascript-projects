Operators

There are multiple kinds of operators. Arithmetic, Assignment, Comparison, Logical, Bitwise.
* Arithmetic = addition, substraction, division, remainder, multiplication, exponents (two stars)

* Increment (++)
* Decrement (--)
* Assignment is the equals sign (+=). Where x=x+3 is the SAME as x *=3
* Comparison Operators. Greater than, less than. For example, let x = 1; console.log (x>0); will output "true"
*- Relational or Equality operators. Equality is three equal signs ===, not equal is !==, like x !==1 means "check if x is not equal to one"
Strict equality operators is === means have to be same type, same value. '1' vs 1 is NOT equal, because it's a string vs number.

Ternary Operators

example: if a customer has more than 100 points, they are a gold customer, otherwise silver. let points = 110 let type = points > 100 ? 'gold' : 'silver' Start with condition, if true then use first value. console.log(type);

Logical Operators

AND &&
Returns true if both operands are true
Example: you want to build a situation where if both A and B are true, the customer gets something. let highIncome = true let goodCreditScore = true let eligibleForLoan = highIncome && goodCreditScore; console.log(eligibleForLoan); (will output 'true')
OR || if either operand is true, then it outputs true let eligibleForLoan = highIncome || goodCreditScore;
NOT ! let applicationRejected = !eligibleForLoan;
